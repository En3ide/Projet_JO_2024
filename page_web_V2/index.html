Dans le cadre d'un Projet en Licence 2 de l'Université d'Artois, nous avons conceptualisé une base de donnée sur les Jeux Olympiques 2024.

Vous êtes donc actuellement sur l'interface de consultation et le moteur de recherche de cette base de donnée.

Son utilisation est très simple, il vous suffit cliquer sur "Passer en mode recherche".
Puis sur la nouvelle page, sélectionner la page et filtrer les recherches si vous le souhaitez.

Vous pouvez revenir à cette page à tout moment en appuyant sur la maison en haut de la page.
